# 子どもの成長記録ダッシュボード | KidsGrowth - 技術仕様書

## 概要

**サービス名**: Kids Growth Tracker
**バージョン**: 1.5.0
**更新日**: 2026-05-29
**URL**: https://appadaycreator.com/kids-growth-tracker/

お子さまの身長・体重・発達マイルストーン・予防接種を一元管理。成長グラフで発育の推移を可視化する無料の子ども成長記録アプリです。複数のお子さまに対応、データは端末内のみに保存。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 子どもの成長記録ダッシュボード | KidsGrowthは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。

## 変更履歴

### v1.5.0 (2026-05-29)
- **カウプ指数の自動計算・表示**: ヒーローカードの3番目統計をカウプ指数（乳幼児発育指数）に変更。月齢別評価（標準/やせ気味/太り気味）バッジ表示
- **乳幼児健診スケジュール**: 予防接種タブに1ヶ月健診〜3歳健診の自動スケジュール表示機能追加（生年月日から推定日算出・実施日記録対応）
- **次回予防接種推奨日の自動表示**: 生年月日から各ワクチンの推奨接種月齢を計算し「今月推奨」「期限超過」バッジ表示・サマリーカウント表示
- **グラフツールチップ**: 成長グラフの各データポイントをホバー/タップすると日付・数値・メモをポップアップ表示
- **成長記録の編集機能**: 記録一覧に「編集」ボタン追加、モーダルで日付・身長・体重・メモを修正可能
- **思い出の日付指定機能**: 思い出フォームに「記録日」入力フィールドを追加（過去の日付指定可、降順ソート対応）
- **データ管理モーダルに統計表示**: 設定モーダルを開くと現在のデータ件数（子ども数・成長記録・思い出・マイルストーン）を表示
- **FAQのバックアップ方法を更新**: エクスポート機能の使い方に回答を更新
- **モバイルUX改善**: テーブルをカード形式に変換（480px未満）・予防接種リストのレイアウト改善・ヒーロー年齢行に記録件数を表示

### v1.4.0 (2026-05-29)
- **データエクスポート/インポート**: JSON形式でバックアップ・復元機能追加（ヘッダー⚙️ボタン）
- **子どもプロフィール編集/削除**: 子どもピルの✏️ボタンから名前・生年月日・性別の編集と削除が可能に
- **記録削除確認ダイアログ**: 成長記録・思い出の削除前に確認モーダルを表示（誤削除防止）
- **成長記録メモフィールド**: 身長・体重入力フォームにメモ欄を追加（例: 離乳食開始、1ヶ月健診）
- **前回比変化量表示**: ヒーローカードに「+Xcm（前回比）」「+Xkg（前回比）」を表示
- **利用者の声修正**: 他サービス流用のテキストをKidsGrowth固有の内容に書き換え
- **SVGグラフレスポンシブ対応**: viewBox+100%幅でモバイルでも見切れなくなる
